Джерело даних для Take Profit
📊 Дані з TradingView:
TP буде встановлено на основі даних з JSON сигналу TradingView:

• tpLevels - множинні рівні TP

Якщо в JSON немає TP, використається Збалансована стратегія.

Якщо в JSON немає SL, використається Нормальний SL (2.0%).
