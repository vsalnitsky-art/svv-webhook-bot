## [1.0.0] - 2024-11-XX

### Добавлено
- 🎉 Начальный релиз проекта
- ⚡ Автоматическое выполнение сделок через webhook
- 📊 Трейлинг стоп-лосс для защиты прибыли
- 🎯 Разделенные тейк-профиты (Split TP)
- 📈 Мониторинг активных позиций в реальном времени
- 📉 Расчет RSI и рыночного давления
- 📊 Подробная аналитика P&L
- 🔐 Защита API ключей с помощью шифрования Fernet
- 🌐 Веб-интерфейс для мониторинга (Scanner и Report)
- 🗄️ SQLite база данных для хранения истории
- 🚀 Поддержка деплоя на Render
- 🔄 Автоматическая синхронизация закрытых позиций
- 📱 Адаптивный веб-интерфейс

### Функционал бота
- Автоматический расчет размера позиции на основе процента риска
- Поддержка кредитного плеча до 20x
- Адаптивный трейлинг стоп-лосс в зависимости от актива
- Двухступенчатый тейк-профит (50%/50%)
- Мониторинг активных позиций каждые 10 секунд
- Расчет RSI для каждой активной позиции
- Расчет рыночного давления (Volume Pressure)

### Аналитика
- График изменения P&L
- Win rate статистика
- Топ-5 монет по прибыльности
- Детальная история всех сделок
- Разделение статистики по Long и Short позициям
- Фильтрация данных по периодам (7 дней, 30 дней, кастомный)

### Безопасность
- Шифрование API ключей через Fernet
- Поддержка как зашифрованных, так и обычных ключей
- Защита от утечки ключей через переменные окружения
- Fallback на локальные ключи для разработки
